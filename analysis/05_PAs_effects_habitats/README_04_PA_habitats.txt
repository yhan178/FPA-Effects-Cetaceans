This folder contains four files.


（1）'analysis_initial_habitat_conditions_PAs_vs_nonPAs.Rmd' and (2) a nb.html version of the file
	This file contains code to test whether the PAs had better initial habitat conditions 
than the non-PAs. It uses data in the files 'seg_wise_floodplain_ndvi.csv' and 'seg_wise_urban.csv' in folder 'data_preparation\'.
	It generates statistical outputs and produces Fig.5a-c in the main text.

(3) 'analysis_habitat_changes_PAs_vs_nonPAs.Rmd' and (4) a nb.html version of the file
	This file tests whether PAs had more river segments with stable habitat conditions than the non-PAs.
	It uses data in the files 'floodplain_area_regression_results.csv', 'cropland_perc_regression_results.csv', 'urban_cluster_results.csv' in folder 'trend_analysis_habitats\'.
	It generates statistical outputs and produces Fig.5d-f in the main text.
